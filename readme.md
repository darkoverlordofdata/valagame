# ValaGame

Think Xna ported to Vala, using Corange initialize SDL with OpenGL.

### Install & Run

    git clone https://github.com/darkoverlordofdata/valagame
    cd valagame
    doran install
    mkdir build
    ./configure
    cd build
    make
    cd ../src
    ./demo


### Dependencies

    You need node.js and doran:

    npm install -g doran-cli

        or

    yarn global add doran-cli
